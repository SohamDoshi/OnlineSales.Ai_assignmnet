Make sure to have the pandas and openpyxl libraries installed before running the script. 
You can install them using pip:
   Pip install pandas openpyxl
To connect with the database(MySQL)
   pip install mysql-connector-python
Open CMD or Terminal and execute above command and make sure you have Python installed on your machine.

configer Your MySQL database details in task1_SQL.py file (database name, username, password)

After that open CMD or Powershell inside Task1_SQL folder/direcotry (shift + right click to open CMD or Powershell in that direcotry)
and simply execute this command
   python task1_SQL.py
After sucessfully executing this command you will get OutPut folder inside same direcotry inside that you will find "report.xlsx" file with result.
